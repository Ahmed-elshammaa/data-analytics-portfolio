# Sales Performance Dashboard

This project analyzes sales performance using a Power BI dashboard.

## Objectives
- Track overall sales and profit
- Analyze monthly sales trends
- Compare sales by category
- Analyze profit by region
- Identify top 10 products by sales

## KPIs
- Total Sales
- Total Profit
- Profit Margin %
- Number of Orders

## Tools Used
- Power BI
- DAX
- Excel (CSV data)

## Key Insights
- Sales peak in November and December.
- Technology is the top-performing category by sales.
- The East region generates the highest profit.
- A small number of products contribute significantly to total sales.
